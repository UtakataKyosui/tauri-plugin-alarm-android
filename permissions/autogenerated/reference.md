## Default Permission

Default permissions for the alarm plugin. Grants access to all alarm management commands.

#### This default permission set includes the following:

- `allow-set_alarm`
- `allow-cancel_alarm`
- `allow-list_alarms`
- `allow-check_exact_alarm_permission`
- `allow-open_exact_alarm_settings`
